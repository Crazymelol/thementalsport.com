# Brand Voice

## Overview
A brand voice is a small set of consistent personality traits applied to every word — it stays fixed while tone flexes to context. Codify it so anyone can write on-brand.

## Framework
1. **Pick 3-4 voice attributes** — e.g., "direct, warm, expert, irreverent." Each with a one-line definition.
2. **Define tone range** — how voice shifts across situations (celebratory vs. apologetic) while attributes hold.
3. **Set vocabulary** — preferred terms, signature phrases, words to avoid.
4. **Write paired examples** — same message on-brand vs. off-brand.
5. **Encode do/don't lists** — concrete word swaps.

## Applying It
- For each attribute, ask "does this sentence sound [attribute]?" before publishing.
- Read aloud — if it doesn't sound like a real person from the brand, rewrite.
- Voice is consistent across channels; format adapts.

## Quick Reference
| Element | Define as | Example |
|---------|-----------|---------|
| Attributes | 3-4 traits | Direct, warm, expert |
| Vocabulary | Use / avoid | "you" not "users" |
| Sentence style | Length, rhythm | Short. Punchy. |
| Reading level | Grade target | Grade 6-8 |
| Do | Active, specific | "We fixed it." |
| Don't | Jargon, hedging | "We're working to optimize." |

## Common Mistakes
- Listing generic adjectives ("friendly, professional") with no definition or examples.
- Confusing voice (fixed) with tone (situational).
- No off-brand examples — hardest part is showing what NOT to do.
- Corporate jargon and hedging that erase personality.
- Voice that no real person could speak aloud.
- Not enforcing it — a doc nobody references drifts within weeks.
