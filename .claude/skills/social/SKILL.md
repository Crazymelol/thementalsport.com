# Organic Social Media Content

## Overview
Each platform rewards native behavior — write for how people actually consume on that feed, not a generic "post" copied everywhere. The first line/3 seconds decide whether anyone sees the rest.

## Hook-First Structure
1. **Hook** — stop the scroll: tension, contrast, number, bold claim, or open loop.
2. **Payoff** — deliver the promised value fast; no throat-clearing.
3. **Depth** — one idea, expanded with specifics, story, or steps.
4. **CTA / loop** — invite a reply, save, share, or follow; ask a real question.

## Platform Principles
- **X**: punchy, opinionated. Lead with the take. Threads = hook tweet + one idea per tweet. Whitespace and line breaks. Reply to grow.
- **Instagram**: visual-first. Carousels for "save-worthy" frameworks (cover = hook). Reels for reach. Caption = mini-story + CTA. Hooks on-screen in first frame.
- **TikTok**: spoken hook in first 2s, native/unpolished, trends and sounds, fast cuts, talk to camera. Watch-time and rewatches drive reach.
- **LinkedIn**: hook line + line break before "see more." Personal story → lesson → takeaway. Plain language, no buzzwords. Pro but human.

## Quick Reference
| Platform | Best format | Ideal length | Primary metric |
|----------|-------------|--------------|----------------|
| X | Thread / one-liner | 1 idea/tweet | Replies, reposts |
| Instagram | Carousel / Reel | 5-10 slides | Saves, shares |
| TikTok | Short video | 15-34s | Watch time |
| LinkedIn | Text + image | 1300-1900 char | Comments |

## Common Mistakes
- Cross-posting identical copy to every platform with no adaptation.
- Burying the hook under context or pleasantries.
- Multiple ideas per post — dilutes the save/share trigger.
- Vague CTAs ("thoughts?") instead of a specific question.
- Polished/corporate tone where native and raw wins (TikTok, X).
- Chasing virality over consistency; posting cadence beats one-off hits.
