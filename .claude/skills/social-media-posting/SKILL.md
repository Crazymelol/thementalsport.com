# Social Media Posting Workflow

## Overview
Consistency and systems beat sporadic brilliance — a repeatable cadence plus repurposing keeps every channel fed without creating from scratch each time.

## Workflow
1. **Plan** — set a weekly cadence per platform; map slots to content pillars in a calendar.
2. **Create the anchor** — produce one substantial piece (long video, thread, post).
3. **Repurpose** — atomize the anchor into platform-native cuts (clips, carousels, quotes, threads).
4. **Adapt captions** — rewrite per platform; never paste identical copy.
5. **Schedule** — batch and queue via scheduler; set platform-appropriate times.
6. **Cross-post** — stagger, don't simultaneously dump the same asset everywhere.
7. **Engage** — reply in the first 30-60 min after publishing to boost reach.

## Captions & Hashtags
- Caption carries the hook even when the visual is the star.
- Hashtags: IG 3-8 relevant + niche; LinkedIn 3-5; X 0-2; TikTok 3-5 trend+niche.
- Avoid generic mega-tags (#love) — they bury you.

## Quick Reference
| Platform | Cadence | Hashtags | Best window |
|----------|---------|----------|-------------|
| X | 1-4x/day | 0-2 | Weekday AM/noon |
| Instagram | 3-5x/week | 3-8 | Lunch, evening |
| TikTok | 1x/day | 3-5 | Evening |
| LinkedIn | 3-5x/week | 3-5 | Tue-Thu AM |

## Common Mistakes
- True simultaneous cross-posting (looks spammy, platforms throttle).
- Identical captions everywhere instead of native rewrites.
- Posting then disappearing — ignoring the early engagement window.
- Hashtag stuffing or irrelevant trending tags.
- No calendar — reactive, inconsistent publishing.
- Repurposing as lazy re-uploads instead of reformatting per platform.
