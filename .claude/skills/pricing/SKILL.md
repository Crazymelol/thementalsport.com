# Pricing Strategy

## Overview
Price to the value the customer perceives, not to cost. Structure (anchors, tiers, the unit you charge by) often moves revenue more than the raw number.

## Framework
1. **Pick a value metric** — charge by the unit that scales with customer value (seats, usage, orders). Aligns price with value received.
2. **Estimate willingness-to-pay** — surveys (Van Westendorp), interviews, competitor benchmarks, and tests — not gut.
3. **Anchor high** — present a premium option first so mid-tier feels reasonable (decoy/center-stage effect).
4. **Design tiers (Good/Better/Best)** — 3 tiers, highlight the middle, gate features by segment, name them clearly.
5. **Psychological tactics** — charm pricing ($X9), bundling, annual discount framing, reducing options to avoid choice paralysis.
6. **Test and iterate** — raise prices on new cohorts and measure conversion + revenue.

## Quick Reference
| Lever | Effect |
|-------|--------|
| Anchoring | High first option lifts perceived value |
| Decoy tier | Pushes choice to target plan |
| Charm ($X9) | Feels cheaper at glance |
| Annual framing | "$X/mo billed yearly" lowers friction |
| Value metric | Scales revenue with usage/value |
| 3 tiers | Center-stage middle plan |

## Common Mistakes
- Cost-plus pricing that ignores perceived value.
- Underpricing — leaves money on the table and signals low quality.
- Too many tiers/options causing paralysis.
- No value metric, so big customers pay the same as small ones.
- Competing only on price (race to the bottom).
- Never testing or raising prices.
- Hidden fees that destroy trust at checkout.
